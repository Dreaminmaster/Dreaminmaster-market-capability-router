# Roadmap

## v0.1 Local Reference

- 规则型摩擦诊断；
- 能力路由；
- 查询网格；
- 平台语言词典；
- 候选和风险评分；
- CLI、Skill、测试和评测。

## v0.2 Structured LLM Adapter

- 兼容 OpenAI-compatible API、本地 LM Studio 和 Ollama；
- JSON Schema 约束输出；
- 规则与模型的冲突处理；
- 证据追踪。

## v0.3 Screenshot Review

- 搜索结果截图；
- 商品详情、价格表和聊天图；
- 图片文字与布局理解；
- 候选批量归一化。

## v0.4 Assisted Browser

- 用户授权下的多查询搜索；
- 候选采集与对比；
- 不自动付款和提交敏感信息；
- 可审计的动作记录。

## v0.5 Knowledge Assets

- 问题—能力—职业—服务图谱；
- 平台语言版本管理；
- 服务交付标准；
- 结果反馈和查询效果统计。

## v1.0 Cross-platform Capability Router

- 闲鱼、淘宝、小红书、行业论坛、自由职业平台、正规机构和官方渠道的统一路由；
- 地区、语言和规则适配；
- 可插拔策略与组织级审计。
