# Example: renovation quote

User: 装修公司发来一份报价单，我担心价格虚高和后续增项。

Expected direction:

- Friction: verification + skill.
- AI/tool: parse and flag obvious duplicates or missing units.
- Professional/market: independent cost estimator, supervisor, or experienced foreman for review.
- Self: confirm measurements, negotiate, and sign.
- Service level: review / second opinion, not full outsourcing.
- Queries: 装修报价审核、造价员家装复核、监理远程看单、装修增项排雷。
- Risks: reviewer affiliated with the same contractor; unclear deliverable; placeholder price.
